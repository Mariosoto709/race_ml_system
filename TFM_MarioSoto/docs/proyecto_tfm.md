# Proyecto TFM - Mario Soto

## Título provisional
Análisis e interpretación automatizada de eventos deportivos con aplicación de reglas y clasificación contextual.

## Resumen
Este proyecto aborda el procesamiento automático de eventos deportivos en tiempo real, con foco en extraer y aplicar reglas específicas y enriquecer los datos de los participantes mediante inferencias contextuales como el sexo aparente.
